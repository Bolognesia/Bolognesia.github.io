Discovery

Custom Instruments New Zealand is a new alliance between 3 musicians looking to expand their hand-crafted instruments business: Keith MacMillan, Davy Stuart and Paddy Burgin. They decided to create a new website that offers different services and information about their music and performance. Website requires a new brand and look that is aligned with their style: craft instruments for the music enthusiastic.

Brief

Develop a new website from zero, design a new logo, brand and fresh look. Website will be build in SilverStripe CMS and will allow to the users to navigate through services, different types of instruments, blog and contact the clients asking questions and making instruments requirements. Website must be responsive.

As a first step. Each client old website was content audited, and organised in order to create a new Information Architecture for the new website and low and high fidelity wireframes. For detailed information about this process, please check the following link: Custom Instrument NZ Documentation

Deliverables for this projects include:

  • Website documentation
  • Launch of website
  • Technical documentation
  • User manual
  • User testing (functionality)

Methods

The website will be build based on:

  • LAPP package (Linux, Apache, PHP and PostgresSQL)
  • Server instance AWS
  • SilverStripe CMS, parent-child pages approach
  • SilverStripe Add-ons (reCAPTCHA, forms, blog, testimonials)
  • Composer for dependencies
  • NoIP for the domain
  • FTP Program (WinSCP for windows and Cyberduck for Mac)
  • Security elements like reCAPTCHA, multifactor authentication, HTTPS and SSL certification
  • AWS backup
